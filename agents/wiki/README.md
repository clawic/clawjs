# Agent Wiki

Internal knowledge base for maintainers and coding agents.

Use this folder for operational learnings that should survive across debugging sessions, especially when the correct behavior is shaped by local runtime state, host setup, or prior incidents.

## How To Use This Wiki

- Review the relevant pages here before changing host-dependent behavior or debugging a repeated issue.
- Prefer adding focused pages by topic instead of growing `AGENTS.md` with incident-level detail.
- Record discoveries that would prevent future false assumptions, flaky repro steps, or repeated dead ends.

## Current Pages

- `openclaw.md`: host-dependent OpenClaw runtime, auth, cleanup, and validation learnings.
