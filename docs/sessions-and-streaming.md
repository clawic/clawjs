# Sessions and Structured Stream Events

This page is kept as a compatibility alias for older links. The
canonical reference now lives in [Conversations](/conversations).

## Quick Reference

- Conversation transcripts live under `.clawjs/conversations/<session-id>.jsonl`
- `claw.conversations` exposes session CRUD, title generation, session
  search, structured event streaming, and raw chunk streaming
- `claw sessions stream --events` emits the same normalized event model
  as the Node API
